# TechBridge

O **TechBridge** é um protótipo de plataforma web educativa voltada para inclusão digital. O projeto foi desenvolvido com o objetivo de ajudar pessoas iniciantes a aprenderem conceitos básicos de tecnologia, uso de e-mail, criação de currículo, segurança online e inteligência artificial.

## Link do projeto

https://techbridgee.netlify.app/

## Objetivo

Criar uma plataforma simples, acessível e educativa para apoiar pessoas com pouca familiaridade com tecnologia, oferecendo conteúdos introdutórios e práticos sobre ferramentas digitais e temas importantes do mundo online.

## Funcionalidades

- Página inicial apresentando a proposta do projeto.
- Área de conteúdos educativos.
- Aulas sobre tecnologia e inclusão digital.
- Orientações sobre criação de e-mail.
- Orientações sobre criação de currículo.
- Questionários para fixação do aprendizado.
- Conteúdos introdutórios sobre inteligência artificial.
- Interface publicada online via Netlify.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Netlify

## Temas abordados

- Inclusão digital
- Uso básico da internet
- Criação e uso de e-mail
- Elaboração de currículo
- Segurança online
- Inteligência artificial
- Aprendizado digital para iniciantes
